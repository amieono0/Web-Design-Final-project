# SmartCare SL

A modern healthcare website developed for the Web Design Project.

## Features

- Home Page
- About Page
- Services Page
- Doctors Directory
- Doctor Profiles
- Appointment Booking
- AI Symptom Checker
- Patient Card Generator
- Emergency Services
- Notifications & Reminders
- Health Tips
- Contact Page
- Responsive Design

- JavaScript
